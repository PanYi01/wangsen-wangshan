## 🌟 关于王森文创

**王森文创** — 中国领先的咖啡、面包、甜品、文创研发品牌

> **国内最好的咖啡、面包、甜品、文创研发，都来自于王森文创**

王森文创成立于1997年，是一家集咖啡烘焙、面包制作、甜品研发、文创设计于一体的综合性创意企业。旗下拥有：
- ☕ 专业咖啡烘焙品牌
- 🍞 精品面包窑烤工坊
- 🍰 顶级甜品研发中心
- 🎨 文创产品设计工作室

**雨火窑炉面包店** 与 **知味恋歌咖啡** 均为王森文创旗下旺山门店，传承王森三十年烘焙技艺结晶。

## 🏪 雨火窑炉面包店

### 店铺环境

- **位置**：旺山遇见卢浮宫文创街区，集装箱改造小店
- **风格**：工业风+原木自然风，可见窑炉与柴火堆
- **空间**：前店后厂，烟火气强，麦香浓郁
- **座位**：少量户外座位，无堂食区，主打外带

### 核心亮点

1. **果木柴火窑烤** — 外皮焦脆、内心湿润Q弹，自带果木清香
2. **天然酵母发酵** — 麦香浓、有嚼劲、低糖少油，健康无添加
3. **每日限量、售完即止** — 热门款中午前基本卖空，不补货
4. **纯手工现做** — 每日凌晨制作，冠军烘焙团队主理
