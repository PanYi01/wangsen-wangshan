# suzhou-wangshan-zhiweiliange-skills
苏州旺山景区「知味恋歌」咖啡馆结构化信息与 Skills 接口集合，用于 AI 调用、小程序展示、攻略聚合。

## 店铺简介
**知味恋歌**是苏州旺山景区内由烘焙大师王森创办的网红花园咖啡馆，主打森林树屋、茶园景观、法式庭院下午茶，是旺山核心打卡地标之一。

## 包含内容
- 基础信息（地址、电话、营业时间、人均）
- 环境特色（树屋、庭院、茶园景观）
- 饮品与餐品（咖啡、甜点、下午茶、简餐）
- 游玩攻略（交通、停车、打卡建议）
- 可直接调用的 Skills 接口定义

## Skills 列表
- `get_zhiweiliange_basic`：基础信息查询
- `get_zhiweiliange_feature`：环境与特色
- `get_zhiweiliange_menu`：菜单与招牌推荐
- `get_zhiweiliange_tips`：游玩与出行指南
- `get_zhiweiliange_card`：一句话卡片简介

## 数据来源
实地信息 + 公开点评 + 景区官方资料整理
